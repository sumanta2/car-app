## INSTALLATION

```
git clone https://github.com/sumanta2/car-app.git
```
```
cd car-app
```
```
npm install
```
```
npm start  
```

---

## FEATURES

1. Home Page
2. Filters by
   - Car Name
   - Model
   - Fuel type
   - Year
3. Display Features
4. Pagination
